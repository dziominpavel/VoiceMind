# Работа с ReminderParser

## Контекст
`ReminderParser` — unit-tested парсер русских временных фраз.
Работает **offline**. Находит `fireAt` (Instant) и `body` из STT-текста.

## Перед изменениями
1. Прочитать `docs/REMINDER_PARSING.md` — форматы фраз и edge-cases.
2. Найти текущий парсер: ищи `ReminderParser` или похожий класс.

## Добавление паттерна
3. Добавить test-case в unit-тесты парсера **перед** реализацией.
4. Реализовать разбор паттерна.
5. Запустить тесты — всё должно быть зелёным.
6. Проверить, что не сломаны существующие кейсы.

## Ограничения
- **Не** добавлять сетевой парсер — offline MVP.
- Fallback-парсер (OpenRouter) — только фаза 5, если явно указано.
- Поддержка только **ru-RU** в MVP.

## Тестирование
- Запускать `./gradlew test` или тесты через IDE.
- Критичные кейсы: «через 15 минут», «завтра в 9», «в понедельник в 18:30».
