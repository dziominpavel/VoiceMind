## MODIFIED Requirements

### Requirement: Экран настроек — карточка «О приложении»
Карточка «О приложении» ДОЛЖНА показывать только номер версии (без суффикса типа сборки) и имя разработчика.

#### Scenario: Просмотр информации о приложении
- **WHEN** пользователь открывает настройки
- **THEN** в карточке «О приложении» показывается строка «Версия X.Y.Z»
- **AND** ниже — строка «Разработчик Дёмин Павел»
- **AND** не показывается тестовая подсказка и не показывается тип сборки (debug/release)

---

### Requirement: Главный экран — предпросмотр ближайших
Каждый элемент предпросмотра ближайших напоминаний ДОЛЖЕН показывать краткую метку даты под временем.

#### Scenario: Есть предстоящие напоминания
- **WHEN** на главном экране есть SCHEDULED напоминания
- **THEN** под каждым временем (HH:mm) в списке предпросмотра показывается метка даты
- **AND** метка использует `labelSmall` и цвет `TextMuted`
- **AND** метка показывает «сегодня», «завтра», «послезавтра» или «d MMM»
- **AND** метка обрезается (`maxLines = 1`), чтобы не ломать высоту строки 64dp

---

### Requirement: Список напоминаний — отступ в карточке
Карточка предстоящего напоминания ДОЛЖНА иметь вертикальный отступ между иконкой режима доставки и чекбоксом «выполнено».

#### Scenario: Вкладка Предстоящие
- **WHEN** пользователь открывает вкладку Предстоящие
- **THEN** между иконкой режима доставки и чекбоксом есть отступ 4dp
- **AND** иконка и чекбокс не перекрываются друг с другом
