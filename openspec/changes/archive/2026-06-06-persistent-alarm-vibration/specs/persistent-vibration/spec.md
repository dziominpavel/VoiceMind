## ADDED Requirements

### Requirement: Повторяющийся паттерн вибрации
При срабатывании напоминания в режиме VIBRATE_ONLY система ДОЛЖНА воспроизводить повторяющийся паттерн вибрации. При срабатывании в режиме ALARM система ДОЛЖНА воспроизводить тот же паттерн, но только если глобальная настройка `useVibration` включена.

#### Scenario: Паттерн alarm-стиля в VIBRATE_ONLY
- **WHEN** напоминание с режимом VIBRATE_ONLY срабатывает
- **THEN** устройство вибрирует по паттерну `longArrayOf(0, 500, 200, 500, 200, 500, 200, 500)` с повтором с начала
- **AND** амплитуда соответствует уровню `DEFAULT_AMPLITUDE` (или максимально доступной на устройстве)

#### Scenario: Паттерн alarm-стиля в ALARM с вибрацией
- **WHEN** напоминание с режимом ALARM срабатывает
- **AND** настройка `useVibration` установлена в `true`
- **THEN** устройство вибрирует по тому же паттерну с повтором с начала
- **AND** одновременно воспроизводится звук будильника

#### Scenario: ALARM без вибрации
- **WHEN** напоминание с режимом ALARM срабатывает
- **AND** настройка `useVibration` установлена в `false`
- **THEN** звук будильника воспроизводится
- **AND** вибрация НЕ запускается

### Requirement: Автоматическая остановка вибрации
Вибрация ДОЛЖНА автоматически прекращаться не позднее чем через 60 секунд после начала, если пользователь не остановил её раньше.

#### Scenario: Таймер авто-остановки
- **WHEN** напоминание с режимом ALARM или VIBRATE_ONLY срабатывает
- **THEN** вибрация продолжается до 60 секунд
- **AND** по истечении 60 секунд вибрация останавливается автоматически

### Requirement: Ручная остановка вибрации
Пользователь ДОЛЖЕН иметь возможность остановить вибрацию до истечения 60 секунд через любое доступное действие.

#### Scenario: Остановка через действия уведомления
- **WHEN** пользователь нажимает «Готово», «Отложить» или «Отменить» на уведомлении
- **THEN** вибрация немедленно прекращается

#### Scenario: Остановка через AlarmActivity
- **WHEN** пользователь открывает AlarmActivity (для режима ALARM)
- **AND** нажимает кнопку «Стоп» или любое действие на экране
- **THEN** вибрация немедленно прекращается

#### Scenario: Остановка при dismiss уведомления
- **WHEN** пользователь смахивает уведомление
- **THEN** вибрация немедленно прекращается
