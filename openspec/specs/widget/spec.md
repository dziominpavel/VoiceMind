## ADDED Requirements

### Requirement: Виджет ReminderListWidget
Приложение ДОЛЖНО предоставлять домашний виджет размером 4x1 (min 250x55dp), растягиваемый до 5x1 (max 320x72dp).

#### Scenario: Добавление виджета
- **WHEN** пользователь добавляет виджет на домашний экран
- **THEN** отображается список из 3 строк напоминаний
- **AND** справа — кнопка микрофона

### Requirement: Строки виджета
Виджет ДОЛЖЕН показывать до 3 строк: последнее выполненное + ближайшие предстоящие.

#### Scenario: Есть выполненные и предстоящие
- **WHEN** есть и выполненные, и предстоящие напоминания
- **THEN** строка 1 — последнее выполненное (body перечёркнут)
- **AND** строки 2–3 — ближайшие предстоящие

#### Scenario: Только предстоящие
- **WHEN** есть только предстоящие
- **THEN** показываются до 3 предстоящих

#### Scenario: Пустой список
- **WHEN** нет напоминаний
- **THEN** строка 1 показывает "Нет напоминаний"

### Requirement: Клик по body строки
Клик по тексту строки ДОЛЖЕН открывать приложение на деталях напоминания.

#### Scenario: Открыть напоминание
- **WHEN** пользователь нажимает на body строки
- **THEN** открывается приложение с экраном деталей напоминания

### Requirement: Клик по чекбоксу
Клик по чекбоксу ДОЛЖЕН переключать статус напоминания.

#### Scenario: Отметить выполненным
- **WHEN** пользователь нажимает чекбокс предстоящего
- **THEN** статус меняется на DISMISSED
- **AND** виджет обновляется

### Requirement: Клик по микрофону
Клик по кнопке микрофона ДОЛЖЕН открывать голосовой ввод.

#### Scenario: Быстрый голосовой ввод
- **WHEN** пользователь нажимает микрофон на виджете
- **THEN** открывается приложение с активированным голосовым вводом

### Requirement: Цвета виджета
Виджет ДОЛЖЕН использовать прозрачный фон и цвета CLEAR BELL.

#### Scenario: Отображение
- **WHEN** виджет отображается
- **THEN** фон прозрачный
- **AND** текст body белый #FFFFFF, 12sp
- **AND** текст time полупрозрачный белый #B3FFFFFF, 10sp
- **AND** фон микрофона бирюзовый #FF2EC4B6

### Requirement: RemoteViews ограничения
Виджет ДОЛЖЕН использовать RemoteViews со статическим layout.

#### Scenario: Обновление
- **WHEN** данные напоминаний изменились
- **THEN** WidgetReminderProvider обновляет RemoteViews через setInt/setTextViewText
- **AND** нет динамического addView
